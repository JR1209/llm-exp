# 🚀 心理咨询对话生成系统 - 前端界面

## 快速启动

### 方法 1：直接打开（最简单）
双击 `index.html` 文件，在浏览器中直接打开查看界面。

### 方法 2：本地服务器（推荐）
如果需要完整的后端功能，请启动 Python 后端服务器：

```bash
# 在项目根目录运行
python app.py
```

然后访问：`http://localhost:9123`

## 🎨 设计亮点

### 1. 视觉特色
- **深色治愈系主题**：温暖的珊瑚粉 + 专业的深蓝紫
- **流畅动画**：页面加载、悬停、进度等 20+ 种动效
- **玻璃态效果**：半透明卡片 + 渐变边框
- **动态背景**：径向光晕 + 网格动画

### 2. 交互优化
- **拖拽上传**：支持拖放文件，实时视觉反馈
- **实时进度**：渐变流动进度条 + 状态消息
- **实时日志**：自动刷新的实验日志查看器
- **统计展示**：动态数字卡片显示结果统计

### 3. 使用流程

#### 步骤 1：上传问题集
- 点击或拖拽上传 `.json` 或 `.txt` 文件
- 支持多种 JSON 格式
- 即时预览文件内容

#### 步骤 2：运行实验
- 点击 "🚀 开始生成对话" 按钮
- 实时查看进度和日志
- 自动轮询实验状态

#### 步骤 3：下载结果
- 实验完成后查看统计数据
- 点击 "💾 下载 QA 结果集" 获取 JSON 文件
- 包含所有问题的最优对话结果

## 📁 支持的文件格式

### JSON 格式

**格式 1：简单数组**
```json
["问题1", "问题2", "问题3"]
```

**格式 2：对象数组**
```json
[
  {"question": "问题1"},
  {"question": "问题2"}
]
```

**格式 3：嵌套格式**
```json
{
  "questions": ["问题1", "问题2"]
}
```

### TXT 格式
每行一个问题：
```
问题1
问题2
问题3
```

## 🔧 自定义配置

### 修改颜色主题
编辑 `index.html` 中的 CSS 变量：

```css
:root {
    --primary: #FF6B9D;      /* 主色：珊瑚粉 */
    --secondary: #4A5FD9;    /* 辅助色：蓝紫 */
    --accent: #FEC163;       /* 强调色：金色 */
    --bg-dark: #0F1419;      /* 背景：深色 */
}
```

### 调整动画速度
修改动画持续时间：

```css
/* 示例：加快动画 */
.container {
    animation: fadeInUp 0.5s cubic-bezier(0.16, 1, 0.3, 1);
}
```

## 🐛 问题排查

### 问题：样式未加载
- 检查网络连接（Google Fonts）
- 尝试硬刷新（Ctrl + Shift + R）

### 问题：文件上传失败
- 确认文件格式正确（.json 或 .txt）
- 检查文件编码（建议 UTF-8）

### 问题：实验未完成
- 查看实时日志获取详细信息
- 检查后端服务是否正常运行（端口 9123）

## 💡 提示

- 上传前建议先验证 JSON 格式
- 实验运行时可以查看实时日志
- 结果文件包含实验版本号，便于追溯
- 确保后端服务在 9123 端口运行
